# Data Folder

This folder stores the dataset used for training and evaluating the models.

---

## Dataset Source

The dataset is available on Kaggle:  
[Heart Disease Data (UCI)](https://www.kaggle.com/datasets/redwankarimsony/heart-disease-data/data)

---

## Instructions

1. Download the file `heart_disease_uci.csv` from Kaggle.  
2. Place it in this folder (`data/`).  
   The final path should look like: data/heart_disease_uci.csv
